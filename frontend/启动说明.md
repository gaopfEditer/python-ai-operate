# 经济指标日历 - 启动说明

## 快速开始

### 1. 安装依赖

在 `frontend` 目录下运行：

```bash
npm install
```

### 2. 启动开发服务器

```bash
npm run dev
```

服务器将在 `http://localhost:3000` 启动，浏览器会自动打开。

### 3. 构建生产版本

```bash
npm run build
```

构建后的文件将在 `dist` 目录中。

## 功能说明

### 显示的经济指标

1. **美联储利率决议（FOMC）** - 红色
   - 每年8次，通常在1月、3月、5月、6月、7月、9月、11月、12月的周三

2. **鲍威尔新闻发布会** - 深红色
   - 紧随FOMC后30分钟

3. **非农就业数据（NFP）** - 蓝色
   - 每月第一个周五

4. **美国CPI** - 紫色
   - 每月1次，通常在12-15日之间的周三

5. **PPI（生产者物价指数）** - 深紫色
   - 每月1次，通常在CPI前一天

6. **ISM制造业PMI** - 青色
   - 每月第一个工作日

7. **ISM非制造业PMI** - 青色
   - 每月第三个工作日

8. **零售销售** - 橙色
   - 每月第2个周五

9. **初请失业金人数** - 橙红色
   - 每周四

### 时间说明

- 所有时间已自动转换为**北京时间**
- 自动处理夏令时（3月第二个周日-11月第一个周日）和冬令时的转换
- 夏令时：美东时间+12小时 = 北京时间
- 冬令时：美东时间+13小时 = 北京时间

### 使用技巧

- 点击日历上的事件可查看详细信息
- 支持月视图、周视图、日视图切换
- 使用导航按钮切换月份
- 不同颜色代表不同类型的经济指标
