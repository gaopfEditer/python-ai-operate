## 工作流程1：爬取 → 生成 → 发布

```mermaid
flowchart TD
    Start1([开始流程1]) --> ReadConfig1[读取配置]
    ReadConfig1 --> ReadTopic1[读取主题列表<br/>main/topic.js]
    
    ReadTopic1 --> Loop1{遍历每个主题}
    Loop1 -->|有主题| Generate1[生成文章<br/>generate_article_by_topic]
    Loop1 -->|无主题| End1([结束])
    
    Generate1 --> AI1[AI生成内容<br/>主题/要求/风格/字数]
    AI1 --> Save1[保存文章<br/>output/articles/]
    
    Save1 --> Publish1[发布文章<br/>publish_content]
    Publish1 --> Login1[登录Typecho]
    Login1 --> Post1[发布到博客]
    Post1 --> Result1[返回文章链接]
    
    Result1 --> Loop1
    
    style Generate1 fill:#fff4e1
    style Publish1 fill:#e8f5e9
```

## 工作流程2：爬取 → 评论

```mermaid
flowchart TD

    PostComment2 --> Result2[记录评论结果]
    
    Result2 --> Loop2
    
    style Crawler2 fill:#e1f5ff
```


## 模块详细流程图

### 信息抓取模块详细流程

```mermaid
flowchart LR
    StartC([开始抓取]) --> LoadConfig[加载配置<br/>平台列表/关键词]
    LoadConfig --> SelectPlatform{选择平台}
    
    SelectPlatform --> P1[知乎]
    SelectPlatform --> P2[微博]
    SelectPlatform --> P3[抖音]
    SelectPlatform --> P4[其他平台...]
    
    P1 --> Fetch[抓取数据]
    P2 --> Fetch
    P3 --> Fetch
    P4 --> Fetch
    
    Fetch --> FilterC[关键词过滤<br/>frequency_words.txt]
    FilterC --> Format[格式化数据]
    Format --> GenerateHTML[生成HTML报告]
    GenerateHTML --> PushNotify[推送通知]
    PushNotify --> SaveFile[保存文件<br/>output/日期/]
    SaveFile --> EndC([结束])
    
    style Fetch fill:#e1f5ff
    style FilterC fill:#fff4e1
```

### 文章生成模块详细流程

```mermaid
flowchart TD
    StartG([开始生成]) --> Input[输入参数<br/>topic/requirements<br/>platform/content_type<br/>word_count/style]
    
    Input --> BuildPrompt[构建AI提示词]
    BuildPrompt --> CallAI[调用AI接口<br/>通义千问]
    
    CallAI --> Parse[解析AI响应]
    Parse --> Validate{验证内容}
    
    Validate -->|有效| FormatG[格式化Markdown]
    Validate -->|无效| Retry{重试?}
    
    Retry -->|是| CallAI
    Retry -->|否| Error[返回错误]
    
    FormatG --> SaveG[保存文章<br/>output/articles/]
    SaveG --> EndG([生成完成])
    
    style CallAI fill:#fff4e1
    style Validate fill:#e8f5e9
```

### 文章发布模块详细流程

```mermaid
flowchart TD
    StartP([开始发布]) --> LoadArticle[加载文章内容]
    LoadArticle --> InitBrowser[初始化浏览器<br/>Selenium]
    
    InitBrowser --> LoginP[登录平台<br/>输入账号密码]
    LoginP --> CheckLogin{登录成功?}
    
    CheckLogin -->|否| ErrorP[返回错误]
    CheckLogin -->|是| Navigate[导航到编辑页面]
    
    Navigate --> FillTitle[填写标题]
    FillTitle --> FillContent[填写内容<br/>Markdown转HTML]
    FillContent --> SetTags[设置标签]
    SetTags --> Submit[提交发布]
    
    Submit --> CheckPublish{发布成功?}
    CheckPublish -->|是| GetLink[获取文章链接]
    CheckPublish -->|否| ErrorP
    
    GetLink --> CloseBrowser[关闭浏览器]
    CloseBrowser --> EndP([发布完成])
    
    style LoginP fill:#e8f5e9
    style Submit fill:#e8f5e9
```
