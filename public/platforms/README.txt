# 平台操作提示词文件夹

## 说明
此文件夹包含各个平台的特定发布提示词文件，每个平台有独立的发布规则和API配置。

## 平台文件命名规范
- 微博：weibo_prompt.txt
- 知乎：zhihu_prompt.txt
- 小红书：xiaohongshu_prompt.txt
- 抖音：douyin_prompt.txt
- 微信公众号：wechat_prompt.txt
- 其他平台：{platform_name}_prompt.txt

## 平台提示词文件结构
每个平台提示词文件应包含：
1. 发布API接口和认证方式
2. 内容格式要求（字数、图片尺寸、视频格式等）
3. 发布参数配置（可见性、标签、话题等）
4. 发布频率限制
5. 审核规则和注意事项
6. 最佳实践建议

## 使用方式
在执行发布操作时，根据目标平台加载对应的提示词文件，使用其中的规则进行内容发布。
