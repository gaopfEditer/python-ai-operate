# 平台操作提示词文件夹

## 说明
此文件夹包含各个平台的特定评论提示词文件，每个平台有独立的评论规则和操作流程。

## 平台文件命名规范
- 微博：weibo_prompt.txt
- 知乎：zhihu_prompt.txt
- 小红书：xiaohongshu_prompt.txt
- 抖音：douyin_prompt.txt
- 贴吧：tieba_prompt.txt
- 其他平台：{platform_name}_prompt.txt

## 平台提示词文件结构
每个平台提示词文件应包含：
1. 登录方式和API接口
2. 评论格式要求（字数、表情、链接等）
3. 评论频率限制
4. 审核规则和敏感词
5. 评论策略建议
6. 异常处理方案

## 使用方式
在执行评论操作时，根据目标平台加载对应的提示词文件，使用其中的规则进行评论发布。
