# 平台操作提示词文件夹

## 说明
此文件夹包含各个平台的特定提示词文件，每个平台有独立的提示词配置。

## 平台文件命名规范
- 微博：weibo_prompt.txt
- 知乎：zhihu_prompt.txt
- 小红书：xiaohongshu_prompt.txt
- 抖音：douyin_prompt.txt
- 微信公众号：wechat_prompt.txt
- 其他平台：{platform_name}_prompt.txt

## 平台提示词文件结构
每个平台提示词文件应包含：
1. 平台特点说明
2. 内容格式要求
3. 字数限制
4. 标签/话题规范
5. 审核规则
6. 最佳实践

## 使用方式
在创作内容时，根据目标平台加载对应的提示词文件，确保生成的内容符合平台规范。
