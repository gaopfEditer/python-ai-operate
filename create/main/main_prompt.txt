# 创作模块主流程提示词

## 流程说明
此文件用于串联整个创作流程，从数据输入到内容输出的完整流程。

## 主流程步骤
1. **数据接收**
   - 接收来自爬虫模块的热点数据
   - 验证数据完整性和有效性
   - 解析数据格式

2. **内容分析**
   - 分析热点数据的主题和关键词
   - 识别目标受众和平台特点
   - 确定内容创作方向

3. **内容生成**
   - 调用通义千问（Qwen）AI模型生成初始内容
   - 使用 create/global_prompt.txt 作为系统提示词
   - 将热点数据、平台信息、内容类型作为用户输入
   - 根据平台要求调整格式
   - 优化内容结构和表达
   - 如果生成失败，自动重试（最多3次）

4. **内容审核**
   - 检查内容原创性
   - 验证是否符合平台规范
   - 确保内容质量达标

5. **内容输出**
   - 格式化输出内容
   - 保存到指定位置
   - 传递给发布模块

## 集成点
- 输入：从 crawler 模块接收数据
- 输出：向 public 模块传递内容
- 可选：从 comment 模块获取反馈用于优化

## 错误处理
- 数据格式错误：返回错误信息并记录日志
- 生成失败：重试机制，最多3次
- 审核不通过：标记原因，返回重新生成
